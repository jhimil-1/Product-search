Chatbot Widget Demo

Live Demo

This is a live demo of the chatbot widget. You can interact with it to see how it works.

Features:

  • Text-based product search
  • Image-based product search
  • Responsive design
  • Customizable appearance

How to Use

Add this code to your HTML to include the chatbot widget:

<!-- Include the widget CSS --> <link rel="stylesheet" href="/static/css/widget.css"> <!-- Add Font Awesome for icons (optional) --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css"> <!-- Add the widget container --> <div id="chatbot-widget" data-api-key="YOUR_API_KEY" data-primary-color="#4a6fa5" data-greeting="Hello! How can I help you today?" data-position="bottom-right" data-auto-init="true"> </div> <!-- Include the widget script --> <script src="/static/js/embed.js"></script> <!-- Initialize the widget (optional, only needed if not using auto-init) --> <script> // Only needed if not using data-auto-init="true" document.addEventListener('DOMContentLoaded', function() { ChatbotWidget.init({ apiKey: 'YOUR_API_KEY', primaryColor: '#4a6fa5', greeting: 'Hello! How can I help you today?', position: 'bottom-right', autoInit: true }); }); </script>

Configuration Options:

  • data-api-key: Your API key (required)
  • data-primary-color: Primary color for the widget (default: #4a6fa5)
  • data-greeting: Initial greeting message
  • data-position: Widget position (bottom-right, bottom-left, top-right, top-left)
  • data-auto-init: Whether to initialize the widget automatically (default: true)

Using Your API Key

To use the widget on your website, you'll need an API key. Here's how to get started:

1. Get an API Key

Log in to the admin dashboard and generate a new API key with the appropriate permissions.

2. Include the Widget

<!-- Include the widget CSS -->
<link rel="stylesheet" href="/static/css/widget.css">

<!-- Add the widget container -->
<div id="chatbot-widget"
     data-api-key="YOUR_API_KEY"
     data-primary-color="#4a6fa5"
     data-greeting="Hello! How can I help you today?"
     data-position="bottom-right">
</div>

<!-- Include the widget script -->
<script src="/static/js/embed.js"></script>

3. Or Initialize Programmatically

<script>
document.addEventListener('DOMContentLoaded', function() {
    ChatbotWidget.init({
        apiKey: 'YOUR_API_KEY',
        primaryColor: '#4a6fa5',
        greeting: 'Hello! How can I help you today?',
        position: 'bottom-right',
        autoInit: true
    });
});
</script>
